Troubleshooting

This section addresses difficulties I had while learning to use SpaCy myself, as well as how I went about resolving them.

Powershell isn't installing the 'en_core_web_sm' Language Pack. What can I do to resolve this?

Powershell was able to download the library, but failed to link it with SpaCy.

1. Download the Anaconda Installer

It's possible that the installation is conflicting with libraries that you have already downloaded into Python 3.8. Changing installer to Conda may solve this straight away.

2. If the problem wasn't resolved at Step 1, go to VS Code and select "Add Environment". Select "Conda environment". Name the environment 'troubleshooting' and then save it.

3. Open the Anaconda cmd prompt as an Administrator. Enter 'pip install spacy'

4. Still using Anaconda as an Administrator, Enter 'python -m spacy download en_core_web_sm'. Your troubleshooting environment should now include SpaCy and the English language.
